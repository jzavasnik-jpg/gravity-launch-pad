# Avatar Photos Directory

Copy avatar photos from `gravity-course-creator/frontend/public/avatars/` to this directory.

## Required Structure:

```
public/avatars/
├── male/
│   ├── young_black_man1.jpg
│   ├── Young_caucasian_man1.jpg
│   ├── Middle_aged_black_man1.jpg
│   └── ... (see src/lib/avatarPhotos.ts for complete list)
└── female/
    ├── Young_black_woman1.jpg
    ├── Young_european_woman1.jpg
    ├── Middle_aged_black_woman1.jpg
    └── ... (see src/lib/avatarPhotos.ts for complete list)
```

## Notes:
- Case-sensitive filenames required
- ~60-80 total photos needed
- Currently using placeholder fallback images until photos are added
